# NFT with Physical Item example

This example shows how to get the get shipping information and shipping information registration for NFT with Physical Item.

To start this, install the modules and try out with own ACCESS TOKEN and Fortmatic key.

## How to use

Set your ACCESS TOKEN key at `src/main.ts` from Mint Console.

```shell
% npm i
% npm run dev
```

Open http://localhost:3000 to view it in the browser.
